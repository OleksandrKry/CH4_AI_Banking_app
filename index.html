Get in touch View on GitHub See how it works
01The problem

Hundreds of products. No easy way to compare them.

Every product carries different benefits, fees, and requirements — enough to confuse new and experienced customers alike. Today, getting a real answer means calling the bank, visiting a branch, or digging through long documents and web pages.

GOAL

Make that information accessible instantly — anywhere, anytime. That's the brief this app was built against.

02The solution

An AI consultant that guides, not just answers.

We built an on-device banking consultant that goes beyond support — it walks each customer to the right option in an interactive, personalized conversation. Built for the bank's own customers comparing products on their own, whether they're new or experienced.

Functional

Understands natural, conversational questions
Asks only the follow-ups it actually needs
Recommends from the bank's real product catalog
Remembers context so answers don't repeat

Non-functional

Privacy — all inference stays on the device
Cost — no per-query cloud API bill
Speed — local inference, no network round-trip
Personalization — shaped by this conversation

04Conversation design — flows

The model routes every turn.

The exact routing logic, pulled straight from the app's design system.

HIG

Ask before you answer — but only when the user is actually shopping. No app-side routing: the AI picks exactly one move per turn. Questionnaires and product search are tools it calls — greetings never trigger workflows.

LEGEND You Assistant Questionnaire Catalog search Finish & memory Guardrail
1 · Just talkFriendly text answer. No tools, no cards.
2 · QualifyQuestionnaire tool: 3–6 questions, one per turn.
3 · RecommendCatalog search → answer + ★ confidence-gated cards.
4 · RememberAnswers from the conversation — cards never swap.
answers arrive ▶ recommend ▶ follow-ups loop ▶ Apply → finish
